welcome to EasyAFK
to use simply follow these instructions.
1. make sure you have steam downloaded, logged in, and running.
2. download python https://www.python.org/downloads/
3. run IdleMasterExtended.exe from the idle master folder
4. login to steam via idle master
5. run "script generator.py"
6. enter your steam username
7. right click on the run.bat file in the EasyAFK folder
8. click edit
9. change the idle master directory inside of result.bat
10. save result.bat and run it
